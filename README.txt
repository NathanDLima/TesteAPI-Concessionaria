O projeto ConcessionariaWeb foi feito a partir do desafio de criar, como aplicação simples, uma API em C# que realizasse as operações de um CRUD, possuindo também uma interface gráfica intuitiva de se usar.

Com isso em mente, foi criado a devida API tomando como base uma Concessionaria fictícia, que precisa realizar as operações de adição de um novo veículo, exibição de todos veículos cadastrados, exibição dos detalhes de um veículo em específico, a alteração dos dados de um veículo e a venda do mesmo (exclusão do sistema).

O mesmo foi criado usando a linguagem de programação C#, utilizando do framework ASP.NET CORE 6.0 no padrão MVC para melhor estruturação do projeto, para a criação do modelo e do controlador e suas regras de negócio, assim como foi utilizado JavaScript, especificamente JQuery, com o uso do AJAX para a realização das requisições. Também foi utilizado HTML e CSS, assim como o framework Bootstrap, para a estilização das páginas de forma que o projeto pudesse ter um visual simples e intuitivo em suas páginas de visualização.

Por fim, não houve o uso de banco de dados, visto que o teste teve como ideia o uso da memória local para armazenar as operações.
